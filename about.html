About Nepal Sky

सरल मौसम, सबैको लागि — Weather Explained Simply, for Everyone

Nepal Sky is a free, open-source weather platform that makes meteorological data accessible to everyone in simple, human-readable language. No technical jargon, no advertisements, no tracking — just clear weather information for all of Nepal.

🎯 Our Mission

To provide clear, accurate, and easy-to-understand weather information for all of Nepal — in both Nepali and English — without any technical jargon, advertisements, or paid features. Weather information should be accessible to everyone, from farmers planning their crops to travelers planning their journeys.

📡 Data Sources

Weather data is sourced from multiple reliable providers:

🌍 Open-Meteo 🇳🇵 DHM Nepal 📍 OpenStreetMap

Open-Meteo provides real-time city weather data with no API key required. DHM Nepal provides official meteorological analysis, province forecasts, and station data. Forecasts are updated daily at 8:00 AM and 6:00 PM NST.

💻 Technology

Nepal Sky is built with vanilla HTML, CSS, and JavaScript. No frameworks, no backend servers — just static files served directly to your browser. This makes it fast, lightweight, privacy-friendly, and easy to maintain.

HTML5 CSS3 JavaScript PWA Open-Meteo API DHM API

🔓 Open Source

Nepal Sky is open-source software. The complete source code is available on GitHub. Contributions, bug reports, and feature requests are welcome. Help make weather information better for everyone in Nepal.
