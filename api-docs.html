📡 Nepal Sky API Reference

Weather API Documentation

Complete reference for all weather APIs used by Nepal Sky. We use Open-Meteo for real-time weather data and DHM Nepal for official meteorological analysis and forecasts.

🌍 Open-Meteo Weather API

GET https://api.open-meteo.com/v1/forecast

Free, open-source weather API. No API key required. Returns 3-day hourly and daily forecasts for any coordinates. Used for city weather cards and dashboard forecasts.

✅ No API key · No rate limits · CORS enabled · Open data

Query Parameters

Parameter Type Required Description
latitude float Required Latitude of location (e.g., 27.7172 for Kathmandu)
longitude float Required Longitude of location (e.g., 85.3240 for Kathmandu)
current string Required Current weather variables (comma-separated)
daily string Required Daily forecast variables (comma-separated)
hourly string Optional Hourly forecast variables (comma-separated)
timezone string Required Timezone (e.g., Asia/Kathmandu)
forecast_days integer Optional Number of forecast days (1-16, default 7)

Available Weather Variables

VariableDescription
temperature_2mAir temperature at 2m above ground
weather_codeWMO weather code
relative_humidity_2mRelative humidity at 2m
wind_speed_10mWind speed at 10m (m/s)
apparent_temperatureFeels-like temperature
precipitationPrecipitation amount (mm)
precipitation_probabilityChance of rain (%)
sunriseSunrise time
sunsetSunset time

Example Request

// Get weather for Kathmandu
fetch('https://api.open-meteo.com/v1/forecast?latitude=27.7172&longitude=85.3240¤t=temperature_2m,weather_code,relative_humidity_2m,wind_speed_10m&daily=weather_code,temperature_2m_max,temperature_2m_min,precipitation_probability_max&timezone=Asia/Kathmandu&forecast_days=1')

Example Response

{
  "current": {
    "temperature_2m": 28.5,
    "weather_code": 1,
    "relative_humidity_2m": 65,
    "wind_speed_10m": 3.2
  },
  "daily": {
    "temperature_2m_max": [32.5],
    "temperature_2m_min": [22.1],
    "precipitation_probability_max": [40]
  }
}

🇳🇵 DHM Nepal Official APIs

GET https://dhm.gov.np/mfd/api/country-forecast

Official national weather analysis from Department of Hydrology and Meteorology, Nepal. Updated twice daily at 8:00 AM and 6:00 PM NST. Used in the Meteorological Analysis section.

⚠️ May have CORS restrictions · Official government data · Updated twice daily

Example Response

{
  "analysis_en": "At present, Nepal is under the partial influence of monsoon trough...",
  "analysis_np": "हाल नेपालमा मनसुनको न्यून चापीय रेखा...",
  "en_text_1": "Partly to generally cloudy throughout the country...",
  "np_text_1": "देशभर आंशिकदेखि साधारणतया बदली...",
  "en_text_2": "Partly cloudy in hilly regions...",
  "np_text_2": "पहाडी भूभागमा आंशिक बदली...",
  "user": {
    "name": "Meteorologist Name"
  },
  "issue_date": "2026-06-18T08:00:00+05:45"
}
GET https://dhm.gov.np/mfd/api/three-days-forecast-latest

Returns list of all 3-day forecast bulletins. Use the latest bulletin ID to fetch detailed province-by-province forecasts.

Example Response

[
  {
    "id": 1373,
    "title": "Three Days Weather Forecast",
    "issue_date": "2026-06-18T06:00:00+05:45"
  }
]
GET https://dhm.gov.np/mfd/api/three-days-forecast-info/{id}

Returns detailed 3-day forecast with province-by-province breakdown, meteorological analysis, and forecast maps. Used in the Province Forecast section.

GET https://dhm.gov.np/mfd/api/weather

Returns live weather data from all monitoring stations across Nepal with manual forecasts and observations.

Example Response

{

    "nepali_name": "काठमाडौँ",
    "manual_forecast": [{
      "rain_probability": 40,
      "from_temperature": 26,
      "to_temperature": 28
    }]
  }]
}

📍 Geocoding API

GET https://nominatim.openstreetmap.org/search

OpenStreetMap Nominatim API for searching locations within Nepal. Converts place names to coordinates for weather lookup. Used in the dashboard search.

✅ Free · Rate limited to 1 request/second · Requires User-Agent header

Query Parameters

ParameterTypeDescription
qstringSearch query + "Nepal"
formatstringResponse format (json)
limitintegerMax results (10)
countrycodesstringLimit to Nepal (np)
addressdetailsintegerInclude address details (1)

Example Request

// Search for Pokhara
fetch('https://nominatim.openstreetmap.org/search?q=Pokhara,+Nepal&format=json&limit=10&countrycodes=np&addressdetails=1')

Example Response

[
  {
    "display_name": "Pokhara, Kaski, Gandaki Province, Nepal",
    "lat": "28.2096",
    "lon": "83.9856"
  }
]
